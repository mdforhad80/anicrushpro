Open Source Anime Streaming Platform

A production-ready anime streaming website built with pure HTML, CSS, and JavaScript. No frameworks. No dependencies. Deploy anywhere.

Features

HD Streaming
Watch anime in high quality with smooth playback and minimal buffering.
Airing Schedule
Stay up to date with weekly anime airing schedules from Jikan API.
Advanced Search
Filter by genre, season, year, type, status, and sort options.
Sub & Dub
Switch between subbed and dubbed versions instantly while watching.
Characters & Cast
Explore detailed character lists, voice actors, and staff information.
Fully Responsive
Optimized for mobile, tablet, laptop, desktop, and ultrawide screens.

Deploy on GitHub Pages

1. Create a new repository on GitHub.

2. Upload all 6 HTML files (index.html, anime.html, watch.html, search.html, schedule.html, creator.html) to the repository.

3. Go to Settings > Pages in your repository.

4. Select "Deploy from a branch" and choose "main" branch with "/ (root)" folder.

5. Click Save. Your site will be live at https://yourusername.github.io/reponame.

# Or use Git CLI git init git add . git commit -m "Initial commit" git branch -M main git remote add origin https://github.com/username/repo.git git push -u origin main

Deploy on Cloudflare Pages

1. Go to pages.cloudflare.com and sign in.

2. Click "Create a project" and connect your GitHub repository.

3. Select the repository containing the 6 HTML files.

4. In build settings, set "Build command" to empty and "Build output directory" to /.

5. Click "Save and Deploy". Your site will be live instantly with a custom domain option.

# Direct Upload (no Git needed) # 1. Go to Cloudflare Dashboard > Pages # 2. Click "Upload assets" # 3. Drag and drop all 6 HTML files # 4. Click "Deploy"

HTML5
Semantic markup with accessibility features and SEO optimization.
CSS3

Vanilla JS
