Deployment Guides

🚀 GitHub Pages
Create a new public repository on GitHub.
Upload all 6 HTML files to the root of the repository.
Go to Settings → Pages → Source and select Deploy from a branch.
Choose the main branch and / (root) folder, then click Save.
Your site will be live at https://yourusername.github.io/repo-name/ within minutes.
🌐 Cloudflare Pages
Log in to Cloudflare Dashboard and navigate to Pages.
Click Create a project → Upload assets.
Drag and drop your project folder containing all HTML files.
Cloudflare will auto-deploy and provide a *.pages.dev URL instantly.
Optional: Connect a custom domain in the Pages settings.
