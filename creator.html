✨ About This Project

A premium, Netflix‑quality anime streaming front‑end built entirely with vanilla HTML, CSS, and JavaScript. All data is fetched dynamically from the Jikan API.

  • Responsive design (mobile, tablet, desktop)
  • Cinematic hero slider, trending widgets, search, schedule
  • Cache & rate‑limit management
  • Deployable on GitHub Pages or Cloudflare Pages

🚀 Deployment Guide

GitHub Pages:

  1. Push all HTML files to a repository.
  2. Go to Settings → Pages → Source: main branch.
  3. Your site will be live at https://yourusername.github.io/repo.

Cloudflare Pages:

  1. Connect your GitHub repo to Cloudflare Pages.
  2. Set build command: (none) and output directory: /.
  3. Deploy.
