Deploy on GitHub Pages

1. Create a new repository on GitHub.

2. Upload all 6 HTML files (index.html, anime.html, watch.html, search.html, schedule.html, creator.html) to the repository.

3. Go to Settings → Pages and select the branch you want to deploy (usually main).

4. Your site will be live at https://yourusername.github.io/reponame/.

Deploy on Cloudflare Pages

1. Sign up for a free Cloudflare account at pages.cloudflare.com.

2. Create a new project and connect your GitHub repository, or upload the files directly.

3. Set the build output directory to the root (or /).

4. Cloudflare will automatically deploy your site with a global CDN.

Vanilla JS
